# B.L.U.E. Archive — Complete Preservation

**The B.L.U.E. System** (Broad Learning Universal Education)

An open-source education platform project. This archive preserves everything that could be saved from the original project.

## What Exists vs. What Was Planned

### Working Code
- Login pages at `learn.blue` (Pix, React/TypeScript) — **not archived, external**
- Ripple landing page prototype — **in `04_prototypes/`**
- Brotherhood Project codebase — **external, GPL3.0**

### Documentation
- 22-page roadmap — **in `01_roadmap_screenshots/`**
- Discussion summaries — **in `03_discord_chats/`**
- Tech stack analysis — **in `02_verified_text/`**

### What Was Never Built
- Content creation pipeline
- Verifier/moderation system
- Search infrastructure (using DuckDuckGo placeholder)
- Certification system
- Governance voting mechanism
- Legal entity (501c3 never filed)
- Funding
